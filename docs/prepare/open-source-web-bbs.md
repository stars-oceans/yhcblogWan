# 3款开源的网页端仿掘金轻社区平台

## 1.mlogclub/bbs-go
![mlogclub/bbs-go](https://fc.sinaimg.cn/large/6364aa43gy1hlbpvm15anj20k00adn2s.jpg)

bbs-go是一个使用Go语言搭建的开源社区系统，采用前后端分离技术，Go语言提供api进行数据支撑，用户界面使用Nuxt.js进行渲染，后台界面基于element-ui。

*开源版仅限个人学习使用。可联系作者付费商业版
